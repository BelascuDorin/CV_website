# CV_website
Structures my experience and ongoing work

## How this was developed

Built with AI agents as the primary driver for architecture and decision-making.

**Stack decided via [`/grill-with-docs`](https://github.com/mattpocock/skills)** — a structured interview skill that stress-tests plans against a domain model before writing a line of code. All key decisions (tech stack, site structure, content sections, visual style) were resolved through a grilling session and captured in `CONTEXT.md` as a shared domain glossary.

**Agent skills installed** (`grill-with-docs`, `grilling`, `domain-modeling`, `setup-matt-pocock-skills`) configure Cursor to maintain domain vocabulary and ADRs inline as the project evolves.

**Decisions so far**: Next.js + Tailwind, single scrolling page, light mode, deployed on Vercel. Issues tracked via GitHub Issues.
